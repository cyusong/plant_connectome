Instructions for searching

  1. Use "?" for a matching word (e.g., heat?)
  2. Use "!" for an exact match (e.g., pin1!)
  3. Use "&" for matching and related words
  4. Use "@" for matching gene and genes with shared aliases

Some searching tips:

  1. The search engine is rather simple (at least for now), so a 100% match is required.
  2. Consider using partial search terms to broaden your results (e.g., "CESA" as opposed to "CESA1").
