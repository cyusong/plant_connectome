Composition of PlantConnectome



PlantConnectome was contructed by surveying numerous journals (shown in the pie chart to the right) using GPT.

Unifying more than 100,000 plant studies, PlantConnectome can be used to study genes, metabolites, tissues, and other biological entities, illuminating over 350,000 relationships altogether.

Searching PlantConnectome

PlantConnectome offers three search options:


Term Searches

Search by keywords and gene IDs:

  • CESA1
  • AT4G32410
  • Proline

Author Search

Finds and displays the papers of an author.

Pubmed ID Search
Displays papers of interests by Pubmed ID.

Search Results

All search results with hits will return six results:


Number of Papers

The amount of papers in the database that contain the search query is shown for reference.

Functional Annotations

A collection of functions the search query is associated with is displayed.

Abbreviations

A collection of abbrieviations associated with the search query is displayed.

Entity Relationship Graph

This is a directed graph that offers a visual summary of the various entities in the database that the search query shares relationships with. Each circle (i.e., node) on the graph represents an entity; each arrow (i.e., edge) represents a relationship shared between two entities.

This graph is also interactive and its appearance can be altered by the user via a set of options.

Text Summary

A text summary of the entity relationship graph is also shown for reference. Each paragraph in the text summary corresponds to a separate cluster of nodes in the graph.

Entity Relationship Table
A table summary of the node-edge interactions in the Knowledge Network graph is also shown. This table is downloadable as TSV file, is fully sortable, and has clickable Pubmed ID links.
